Alias fopen: Edición rápida con Nano

6 de diciembre, 2025

Si con fcd solucionamos el problema de movernos entre carpetas, con fopen solucionamos el problema de encontrar qué editar.

A veces sé cómo se llama el archivo (o parte de su nombre), pero casi nunca recuerdo exactamente en qué subcarpeta está. Escribir nano ruta/larga/y/olvidada/config.conf rompe el flujo de trabajo.

La Solución

Este alias busca archivos recursivamente desde donde estoy parado, me deja elegir uno con fzf y lo abre automáticamente en mi editor preferido (Nano).

El Código

Al igual que el anterior, esto va en el .bashrc:

# fopen - Buscar archivo y abrirlo con Nano
fopen() {
  local file
  # -type f: Importante, solo buscamos archivos, no carpetas
  file=$(find . -maxdepth 5 -type f -not -path '*/.*' | fzf +m) && nano "$file"
}

¿Cómo funciona?

  • find .: Empieza a buscar aquí.
  • -type f: Ignora las carpetas, solo quiero archivos editables.
  • nano "$file": Si selecciono algo, lo abre. Si cancelo (ESC), no hace nada.

Nota: Esta configuración la encontré originalmente en algún blog que, lamentablemente, ya no recuerdo. Aunque no puedo dar el crédito exacto al autor, quería mencionarlo porque fue el recurso clave para empezar con fzf Tengo anotados varios alias más y, aunque no los uso con mucha frecuencia, iré compartiendo en futuros artículos.
